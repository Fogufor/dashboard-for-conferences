# Проект "Интерактивный дашборд конференции TED"

## Дашборд: https://datalens.ru/5cpdqw1kxdaip

## Цель проекта: 
На основе данных о ранее проведённых конференциях TED проанализировать ключевые характеристики успешных мероприятий и разработать аналитический дашборд, который поможет заказчику принимать обоснованные решения при выборе спикеров, тем и площадок для организации первой конференции в формате TED.

## Основные задачи дашборда:

1.Отобразить ключевую статистику конференций — количество конференций, выступлений, спикеров и используемых тегов.

2.Проанализировать конференции по странам и отдельным событиям, включая среднюю длительность выступлений, просмотры и аплодисменты.

3.Исследовать выступления — выявить самые популярные темы, самые смешные выступления и топовые теги.

4.Оценить выступающих — определить, какие категории спикеров вызывают наибольшую вовлечённость аудитории.
